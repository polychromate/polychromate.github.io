# 通过远程桌面推黄油的方法

# 使用 iPad 来推黄油

本篇文章主要用来介绍移动终端对 Windows 远程操作，不单独针对 iPad 方面或黄油方面

每次在讨论各家平板的有点的时候，总有人会说 iPad 不能推黄油，其实通过远程桌面功能，还是能够实现 iPad 同样的黄油功能的。

下面以 iPad 为例，演示内容

# 一。将 iPad 和 PC 接入同一局域网内

# 1. 直接将 iPad 和 PC 连接在同一个路由器下即可

路由器不可开启 IP 隔离

# 2.Windows 端开启移动热点服务


Windows10 中自带的移动热点位置

# 3. 同时将 iPad 和 PC 连接上手机热点

记得手机连上 WiFi 或断开流量

# 二。找到电脑的 IP 地址

电脑打开 cmd 输入 ipconfig 命令，查看回执，如果是用 WiFi 连接路由器，选择无线局域网适配器 WLAN，如果是用网线连接，就选择以太网适配器。

电脑的 ipv4 地址

# 三。软件部分

# 1. VNC 软件

vnc 需要电脑和 iPad 都安装软件

# (1) 电脑端（服务器端）

电脑端用浏览器搜索 vnc server，进入 vnc 官网下载 Windows 端服务器软件即可。

搜索结果


下载界面

# (2) 平板端（客户端）

平板端进入应用市场下载即可，其他系统的也可去官网下载

第一个就是

# (3) 客户端设置

点击 vnc 右上角的加号，出现这个 details 界面。address 界面填写第二步获取到的电脑的 IP 地址，name 的地方随便填写，只是用来辨识的。

填写之后点右上角的 done，主界面上出现一个电脑图标，点击进去，出现下图界面。
Windows10 的话，username 和 password 就是你登录电脑用的 outlook 账号密码；Windows7 的话，要手动创建账户。

填完之后，继续 continue，就能看到电脑界面了。

远程桌面视角

之后你就能手搓玻璃推黄油了（滑稽）

# 微软远程桌面

同样的方法，右上角加号输入就好，道理一模一样

微软远程桌面

# 四。补充部分

1. 基本上所有直连的远程软件，设置的地方都是大同小异的，主要就是 IP 地址和账号密码的确定，能知道这两个地方，其他的远程软件，如 sbeam 家的 sbeamlink、nvidia 家的 moonlight 都是可以连接的。
2.vnc 连接无法传输声音，建议带上蓝牙耳机一劳永逸。
3. 微软远程桌面相当于是新建了一个桌面，电脑端是无法操作的，可以远程声音，但是如果网络不好，可能声音是断断续续的。
4. 最后，写这个的目的是给大家一个更多的选择。对我来说也就是躺床上玩游戏的方便。Windows 可选的游戏数量比 krkr 多多了。
